# Ping-Han Huang Website

This folder contains a complete static website ready for GitHub Pages.

## Files

- `index.html` - homepage
- `research.html` - research agenda and themes
- `publications.html` - publications and presentations
- `teaching.html` - current teaching page with syllabus link
- `service.html` - service and leadership
- `cv.html` - online CV page
- `assets/css/styles.css` - visual design
- `assets/js/main.js` - mobile navigation and new-tab handling for external/document links
- `assets/files/Business_Analytics_Using_Simulation_PH_updated.pdf` - course syllabus
- `assets/files/Ping-Han_Huang_CV.pdf` - generated CV PDF from the provided CV text

## Upload instructions

1. Unzip this package.
2. Upload the contents of this folder to the root of your GitHub Pages repository.
3. Commit and push.
4. In GitHub Pages settings, deploy from the branch and root folder that contain `index.html`.

## Updating files later

- Replace `assets/files/Ping-Han_Huang_CV.pdf` with a new CV PDF using the same filename.
- Replace `assets/files/Business_Analytics_Using_Simulation_PH_updated.pdf` with the updated syllabus using the same filename.
- Replace `assets/images/pinghan-huang-headshot.jpg` with a new professional headshot using the same filename.
